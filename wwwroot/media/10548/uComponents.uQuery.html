This would be a breaking change, but in my version here
http://our.umbraco.org/wiki/how-tos/useful-helper-extension-methods-%28linq-null-safe-access%29
I've removed Depth and replaced it with Level and aligned it so root is 0 so it matches xslt

/// <summary>
/// Tell me the level of this node (0 = root)
/// updated from Depth and changed to start at 0
/// to align with other 'Level' methods (eg xslt)
/// </summary>
/// <param name="n"></param>
/// <returns></returns>
public static int Level(this Node n)
{
return n.Path.Split(',').Length-1;
}
